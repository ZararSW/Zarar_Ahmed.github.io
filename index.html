An external domain is a domain name that uses one or more Porkbun hosting products (web hosting, URL forwarding, DNS hosting, email, etc.) even though the domain is still registered at a different registrar.

2FA - Email Based (beta)

Least secure 2FA method but better than nothing, maybe.

You can allow a two factor code to be sent to your account email address. Please note that 2FA via email is considered inherently less secure than other methods such as using an app or a security key. Your account email address must be verified before you can turn on email base 2FA.

Please note that app based 2FA or a security key will override email based 2FA.

2FA - Security Key

Most secure 2FA method.

WebAuthn using security keys is a W3C global standard for secure authentication on the Web supported by all leading browsers and platforms. Porkbun currently supports WebAuthn as a second factor or passwordless login option.

You can read more about WebAuthn at yubico.com/webauthn.

Unrecognized Device 2FA

Fallback 2FA if no other 2FA is enabled.

If you have no other forms of two factor authentication enabled we will check and see if the device you are using has recently been used to log in to your account. If not, we will send you a one time use code to your account email address. This helps prevent logins from hackers that have gotten your credentials via phishing, brute force, or other nefarious means.

Some folks prefer to live life on the edge and want no form of two factor authentication, even this. Every account at Porkbun that has been hacked into has not had 2FA enabled and we highly recommend against turning it off, doing so is at your own risk. We know, it'll never happen to you. However, if you are so inclined to be less secure you can disable this feature.
